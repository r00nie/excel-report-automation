# Excel Report Automation

Automated system for processing Excel files and generating reports using n8n and Python.

## Project Goals
- Process Excel files from email attachments
- Process Excel files from local folder
- Validate and aggregate data
- Generate reports automatically
- Send reports via email

## Tech Stack
- n8n
- Python (FastAPI, pandas)
- Docker
